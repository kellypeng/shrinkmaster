Opening on macOS for the first time

ShrinkMaster isn't yet signed with an Apple Developer ID, so macOS will block the first launch. One-time workaround:

  1. Drag ShrinkMaster.app into your Applications folder
  2. Right-click the app, choose Open (don't double-click yet)
  3. Click Open in the dialog macOS shows

After that, it opens normally. If macOS still refuses, run this in Terminal:

xattr -cr /Applications/ShrinkMaster.app
